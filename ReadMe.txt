SOURIGUES Julien
DRAOU Meryem

Dans notre jeu, nous avons une carte carré qui est une grille et le joueur va commencer dans le coin en haut à gauche et devra aller jusque dans le coin en bas à droite pour finir le jeu.
Pour jouer à ce jeu, les touches utiles sont:
    - les flèches directionnels pour se déplacer dans le donjon.
    - la touche 'Entrée' pour démarrer le jeu puis pour combattre les monstres.
    - la touche 'I' pour ouvrir et fermée l'inventaire en toute situation pendant le jeu.
    - la touche 'A' pour choisir le premier mode combat, puis pour choisir d'utiliser le premier emplacement dans l'inventaire.
    - la touche 'Z' pour choisir le second mode combat, puis pour choisir d'utiliser le second emplacement dans l'inventaire.
    - la touche 'E' pour choisir le troisième et dernier emplacement dans l'inventaire.
Dans notre jeu, nous avons décidé de borner l'inventaire à 3 emplacements et de masquer son contenue car le joueur peut utiliser une potion de poison sans le savoir.
Pour finir le jeu, il faut donc soit arriver à la dernière salle, soit mourir contre un monstre ou en buvant une potion de poison.
